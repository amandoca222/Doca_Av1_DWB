# AV1 - Desenvolvimento Web (API de Música)

Projeto que consome a API TheAudioDB para buscar e exibir álbuns musicais.

## Funcionalidades

- Busca de artistas
- Listagem de álbuns
- Página de detalhes de cada álbum
- Navegação entre páginas
- Uso de URLSearchParams
- Consumo de API com fetch e async/await
- Tratamento de erros e loading

## Tecnologias

- HTML5
- CSS3
- Bootstrap
- JavaScript puro
- Fetch API
- JSON

## Como usar

1. Digite o nome de um artista
2. Clique em "Buscar"
3. Clique em "Ver detalhes" em um álbum
